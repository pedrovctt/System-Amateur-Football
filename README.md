# Amateur Football Ranking System

A simple web app to manage amateur football teams, record matches, and see basic rankings. Built using **PHP**, **MySQL (mysqli)**, **HTML**, and **CSS**. This project is still in development and it's in german. Currently, team and match registration, as well as a basic display of teams and rankings, are implemented.

## Features (Implemented)

- **Team Registration**
  - Add a team with name, city, and founding year.
  - Stored in the `teams` table in the database.

- **Match Registration**
  - Record matches between two teams with scores.
  - Stored in the `spiele` table in the database.

## Planned Features

- Automatic ranking calculation based on wins, draws, and losses.
- Enhanced ranking display (sorted by points, goal difference, goals scored).
- Filters by city or match date.
- Highlight teams with winning streaks.
- View a list of registered teams.
- View a simple ranking table.
- More CSS

## Technology

- **Backend:** PHP + MySQL (PHP extension: mysqli)
- **Frontend:** HTML + a bit of CSS
- Simple forms for team and match registration
- Tables to display teams and rankings
